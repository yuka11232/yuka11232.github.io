# YankeeBaku

## Overview
YankeeBaku is an independent cultural mapping project documenting locations in Baku
connected to American culture and influence, including food, leisure, and informal
cultural spaces.

The website explores everyday cultural exchange rather than official institutions,
presenting culture through places people actually visit.:D

## Purpose
This project was created to better understand how American culture manifests locally
in Baku through urban spaces. It also serves as an experiment in cultural
documentation and public-access research.

## Development Background
The project was initially developed locally without formal documentation or version
control. Early development notes were not preserved.

the project was migrated to GitHub to ensure transparency, structured documentation, and future  development.

## What the Website Includes
- list of locations connected to American culture in Baku
- Short contextual descriptions for each location
- Categorization by type and cultural relevance
- Public, non-commercial access
